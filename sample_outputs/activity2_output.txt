I built a browser game called Cricket Game solo, using HTML5 canvas, where the player swings a bat by hitting the spacebar at the right moment as a ball arcs toward a hit zone. I wrote the ball's pitch using a sine-wave trajectory so it curves unpredictably before lining up with the hit zone, and built a hit-detection check that compares the ball's position against the hit zone's center within a fixed tolerance. I added a difficulty system that increases the ball's base speed in proportion to the player's score once it passes 10, so the game gets harder the longer a streak continues. On a miss, I reset the round and logged the score into a running history panel showing the player's last 10 games. I built it in vanilla JavaScript with the HTML5 Canvas API for rendering and CSS for layout, and deployed it as a live, playable page on GitHub Pages.
