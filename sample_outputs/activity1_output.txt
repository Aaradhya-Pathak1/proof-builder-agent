Activity going in:
I built the Caarya Grand Quiz solo, a full-stack web app with live, global analytics, and presented it at my Midterm Seminar to my Faculty in Charge, Swastibrata Bhattacharyya. I deployed it using static hosting on GitHub Pages. During the seminar I ran a live demo of the quiz engine's rapid-fire timer, a dynamically drawn SVG bell curve, and a score projection matrix. I designed a Bayesian Blending approach so percentile calculations stay fair when only a few people have played a quiz — the app leans on theoretical estimates when data is thin and shifts to live data as more people play. I built the backend using the Firebase REST API, and handled the frontend with browser DOM manipulation and asynchronous JavaScript.
